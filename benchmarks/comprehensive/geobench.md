## 📝 输入输出示例

### 示例1: 地学问答题

**输入（Input）**:
```
问题: 什么是板块构造理论？请简要说明其主要内容。

请给出答案。
```

**期望输出（Expected Output）**:
```
板块构造理论是解释地壳运动的理论，主要内容包括：
1. 地球表面由多个板块组成
2. 板块在软流圈上移动
3. 板块边界有构造活动（如地震、火山）
4. 板块运动导致大陆漂移和海洋扩张
```

### 示例2: 地学计算题

**输入（Input）**:
```
问题: 如果某地的经度是120°E，时区是UTC+8，那么当UTC时间是12:00时，该地的当地时间是多少？

请给出数值答案。
```

**期望输出（Expected Output）**:
```
20:00

计算过程：
UTC+8表示比UTC快8小时
当地时间 = UTC时间 + 8小时 = 12:00 + 8:00 = 20:00
```
