## 📖 评测集介绍

AGIEval（AGI Evaluation）是一个以人为中心的基准测试集，专门设计用于评估基础模型在人类认知和问题解决相关任务中的通用能力。该基准来源于20个官方、公开、高标准的人类考试，包括大学入学考试（如中国高考和美国SAT）、法学院入学考试、数学竞赛、律师资格考试和国家公务员考试等。

### 特点

- **人类中心**: 基于人类考试设计，评估模型的通用能力
- **多样性**: 涵盖多种考试类型，包括大学入学考试、法学院入学考试、数学竞赛、律师资格考试和国家公务员考试等
- **高标准**: 基准测试来源于官方和公开的高标准考试，确保评测的权威性和可靠性
- **综合评估**: 评估模型在语言理解、数学推理、逻辑推理等多个维度的综合表现
- **输入输出**: 输入根据任务类型变化（通常1-5个句子），输出可能是选项、答案或过程（长度可变）

### 适用场景

- 综合能力评估
- 多任务能力评估
- 模型能力全面评估

### 适用模型

- **大模型（>10B）**: 强烈推荐，综合评估需要强大的能力
- **中等模型（1B-10B）**: 可以使用，但可能在某些任务上表现有限
- **小模型（<1B）**: 可以使用，但可能表现有限

## 📖 使用指南

```python
from datasets import load_dataset

dataset = load_dataset("lighteval/agi_eval_en")
```

### 2. 运行评测

#### 使用OpenCompass

```python
from opencompass import OpenCompass

config = {
    'dataset': 'agieval',
    'model': 'your_model',
    'temperature': 0
}

result = OpenCompass.run(config)
```

### 3. 结果分析

- **任务分析**: 分析不同任务的表现
- **维度分析**: 分析不同维度的表现

## 📝 输入输出示例

> **注意**: 以下示例基于AGIEval数据集的典型格式，用于说明评测集的结构。真实题目请从AGIEval数据集获取。

### 示例1: 语言理解任务

**输入（Input）**:
```
问题: 以下哪个选项最能概括这段话的主旨？

[一段文字内容]

A. 选项A
B. 选项B
C. 选项C
D. 选项D
```

**期望输出（Expected Output）**:
```
B. 选项B
```

### 示例2: 数学推理任务

**输入（Input）**:
```
问题: 如果一个数的平方等于16，那么这个数可能是多少？

请给出答案和推理过程。
```

**期望输出（Expected Output）**:
```
4或-4

推理过程：
因为 4² = 16 且 (-4)² = 16，所以这个数可能是4或-4。
```

### 示例3: 逻辑推理任务

**输入（Input）**:
```
问题: 如果所有的鸟都会飞，企鹅是鸟，那么企鹅会飞吗？

请给出答案和推理过程。
```

**期望输出（Expected Output）**:
```
不会。

推理过程：
虽然"所有的鸟都会飞"和"企鹅是鸟"这两个前提都是真的，
但实际上企鹅不会飞。这说明"所有的鸟都会飞"这个前提是错误的。
因此，企鹅作为鸟，不会飞。
```

## 📊 涵盖的考试类型

AGIEval包含以下20个官方考试：

- **大学入学考试**: 中国高考、美国SAT等
- **法学院入学考试**: LSAT等
- **数学竞赛**: 各类数学竞赛题目
- **律师资格考试**: 国家律师资格考试等
- **国家公务员考试**: 各类公务员考试题目

每个考试都包含多个题目，全面评估模型在不同领域和难度级别上的表现。
