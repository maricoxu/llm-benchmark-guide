## 📝 输入输出示例

### 示例1: 柱状图理解

**输入（Input）**:
```
图像: [一张柱状图，显示2020-2024年的销售额]
问题: 哪一年的销售额最高？

请给出答案。
```

**期望输出（Expected Output）**:
```
2023年

根据柱状图，2023年的柱状图最高，表示销售额最高。
```

### 示例2: 流程图理解

**输入（Input）**:
```
图像: [一张流程图，显示数据处理流程]
问题: 这个流程图的最后一步是什么？

请给出答案。
```

**期望输出（Expected Output）**:
```
数据输出

根据流程图，最后一步是数据输出。
```
