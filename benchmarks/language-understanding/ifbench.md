## 📖 评测集介绍

IFBench是用于评估模型在人机交互任务上能力的评测集。

### 特点

- **交互任务**: 评估交互式任务能力
- **多轮对话**: 支持多轮对话交互
- **实际应用**: 贴近实际应用场景

### 适用场景

- 人机交互能力评估
- 多轮对话能力评估
- 交互式任务评估

## 📝 输入输出示例

### 示例1: 多轮对话

**输入（Input）**:
```
用户: 我想订一张从北京到上海的机票。

请给出响应。
```

**期望输出（Expected Output）**:
```
好的，我来帮您订票。请问您希望什么时间出发？
```

**继续对话**:
```
用户: 明天上午

请给出响应。
```

**期望输出（Expected Output）**:
```
好的，我为您查询明天上午从北京到上海的航班...
```
