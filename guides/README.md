## 🚀 快速开始

### 1. 选择评测集

根据你的需求选择合适的评测集：
- 评估模型能力 → Intelligence维度
- 评估推理速度 → Speed维度
- 分析成本效益 → Price维度
- 评估硬件性能 → Hardware维度

### 2. 准备环境

安装评测工具：
```bash
pip install opencompass
# 或
pip install lm-eval
```

### 3. 运行评测

使用评测工具运行评测，获取结果。

### 4. 分析结果

分析评测结果，找出模型优势和改进点。
